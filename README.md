Service-Worker-Middle
======================

Pug(Jade) template on service worker example.

Simple and fast response.

Without dynamic server. (only use static server)


Usage
---------------------

`clone`  

`npm install`  
`bower install`  
`typings install`  
`npm run build`  
`cd ./src/client`  
`tsc`  
`cd ../../src/service`  
`tsc`  
`cd ../../`
`http-server -p 3003 -c1`  

and browse http://localhost:3003/
